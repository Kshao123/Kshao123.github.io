简单搞一题

异或运算符 ^

特性

  1. 交换律：a ^ b ^ c <=> a ^ c ^ b
  2. 任何数与 0 异或，为任何数：0 ^ n => n
  3. 相同的数异或为 0

搞一题

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。
说明：
你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

根据特性 1，所以上面的可以理解为
2 ^ 2 ^ 1 => 0 ^ 1 => 1
